# 01 Virtual Machines

## 01 Create Virtual Network

1 - Set the Name

ie: `demo-vnet`

2 - Set the Address Space

ie: `10.0.0.0/16`

3 - Set the Subnet Name

ie: `demo-subnet`

4 - Set the Address Range

ie: `10.0.1.0/24`

## Create Windows Server 2016

1 - Set the Name

2 - Set the Username and Passwords

3 - Choose the Size

ie: `B1ms`

4 - Select Network

5 - Select Public Inbound Ports

ie: `RDP`

## Create Linux VM

1 - Set the Name

2 - Enter you SSH Public key

3 - Choose the Size

ie: `B1ms`

5 - Select Public Inbound Ports

ie: `SSH`
